# Backend - API REST Consultorio Dental

API REST para gestión de consultorio dental construida con Node.js, Express y MongoDB.

## Instalación

```bash
npm install
```

## Configuración

2. CRea una base de datos en MongoDB nombrada `consultorio_dental`

## Ejecutar el servidor
npm run dev


El servidor estará disponible en `http://localhost:3000`

## Endpoints

Ver la documentación completa en el README principal del proyecto.

## Estructura

```
backend/
├── config/         # Configuración (database.js)
├── controllers/    # Controladores de la API
├── middlewares/    # Middlewares (auth, validation, errorHandler)
├── models/         # Modelos de MongoDB
├── routes/         # Rutas de Express
└── server.js       # Punto de entrada de la aplicación
```

## Tecnologías

- Node.js
- Express.js
- MongoDB
- JWT (JSON Web Tokens)
- Express Validator
- CORS
